The Challenge

Building a fully functional social media platform is a complex architectural challenge. The goal was to replicate the core "addictive" features of Instagram—infinite scroll, instant likes, and real-time updates—within a scalable environment.

I utilized MongoDB for the robust backend to handle user data schemas, while React provided a snappy, app-like experience on the frontend. The interface mimics the native iOS feel using CSS Scroll Snapping and blurred backdrops.
