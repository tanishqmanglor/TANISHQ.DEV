01. The Architecture

Social media applications require handling massive amounts of real-time data. The core challenge was designing a system that remains responsive under heavy load.

The approach: A decoupled MERN architecture. The frontend (React) communicates via REST APIs to a stateless Express backend. I implemented Redux Thunk for asynchronous logic to ensure the UI never freezes during data fetching.

02. Technical Implementation

Security and speed were paramount. I used JWT (JSON Web Tokens) for stateless authentication, meaning the server doesn't need to query the database just to verify a session.

Key Features:

  • Cloudinary Integration: Offloaded image processing and CDN delivery.
  • Optimistic UI: Likes update instantly before server confirmation.
  • Infinite Scroll: Pagination implemented via Intersection Observer.
